Week 1 Lesson Plan
==================

### Subject ###

Basics of a computer and programming

### Central Theme ###

Learning the basics of a computer and programming

### Objective Statement ###

Every child can understand the basics of programming by understanding a few components of a computer.

### Rationale ###

1. Boot up, BIOS and hard drive
2. Operating System
3. Memory
4. Processing - logic
5. Monitor, screen, drawing

### Resources ###

1. [Learn Python The Hard Way](http://learnpythonthehardway.org/book/)
2. Programming Lesson -- covering printing to screen, doing some logic (math, if statement), assigning variables
